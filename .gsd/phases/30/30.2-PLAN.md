# Plan 30.2: Frontend WhatsApp-Clone UI & Media

## Objective
Build the real-time React component for Chama messaging. Members should see a WhatsApp-style UI with message bubbles and be able to attach photos/videos which upload to Firebase Storage before sending via Socket.io.

## Context
- `.gsd/SPEC.md`
- `.gsd/phases/30/RESEARCH.md`
- `frontend/src/pages/chama/core/ChamaDetails.jsx`

## Tasks

<task type="auto">
  <name>Build Real-Time Chat Component</name>
  <files>
    - frontend/src/components/chat/ChamaChat.jsx
    - frontend/src/components/chat/ChamaChat.css
    - frontend/src/pages/chama/core/ChamaDetails.jsx
  </files>
  <action>
    - Create `ChamaChat.jsx` displaying a scrollable list of messages (bubbles aligned left for others, right for the current user).
    - Integrate `SocketContext` to listen for `new_message` events and update the React state instantly.
    - Add a message input bar at the bottom with a text field and an "Attach Media" button.
    - Mount `ChamaChat` as a new tab inside `ChamaDetails.jsx`.
  </action>
  <verify>Run the frontend server and verify the UI renders without crashing.</verify>
  <done>Chat UI is visible and can listen to Socket.io events.</done>
</task>

<task type="auto">
  <name>Implement Media Upload (Firebase Storage)</name>
  <files>
    - frontend/src/components/chat/ChamaChat.jsx
    - frontend/src/services/firebaseStorage.js
  </files>
  <action>
    - Create `firebaseStorage.js` utilizing the existing `firebase-app` config to upload `Blob`/`File` objects to the active bucket.
    - In `ChamaChat.jsx`, when a user selects a photo/video, show a loading spinner, upload to Firebase, retrieve the download URL, and submit the message with `type='image'` or `type='video'` + `media_url`.
    - Ensure the UI renders `<img>` or `<video controls>` if a message contains media.
  </action>
  <verify>Manually upload a test image via the UI.</verify>
  <done>Media uploads to Firebase and renders correctly in the chat bubble.</done>
</task>

## Success Criteria
- [ ] Users can send and receive text messages instantly.
- [ ] Users can attach an image/video, which is uploaded to Firebase.
- [ ] Media renders correctly inside the WhatsApp-style chat bubbles.
